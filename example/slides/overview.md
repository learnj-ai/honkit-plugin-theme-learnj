# LearnJ Theme

### HonKit documentation with Tailwind + shadcn

Note: Press **?** for keyboard shortcuts · **S** for speaker notes

---

## What you get

- Sidebar navigation + search
- shadcn-style **code panels** with copy
- Centered **Mermaid** diagrams
- Dark mode toggle

---

## Architecture

```
Markdown book → HonKit → theme-learnj → static HTML site
```

---

## Code example

```bash
cd example
npm install
npm run serve
```
