# LearnJ HonKit Theme

Welcome to the **LearnJ** theme demo. This example book shows how documentation looks with Tailwind CSS and shadcn/ui design tokens on top of HonKit.

Use the sidebar to explore setup instructions, configuration options, and typography samples. Toggle **dark mode** from the header, resize the window to try the mobile sidebar, and use the search box to find content.

> This site is built from the [`example/`](https://github.com/learnj-ai/honkit-plugin-theme-learnj/tree/main/example) folder in the theme repository and published to GitHub Pages on every push to `main`.
