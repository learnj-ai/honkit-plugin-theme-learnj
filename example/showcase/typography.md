# Typography

HonKit renders standard Markdown into the main content area with Tailwind Typography (`prose`) styling.

## Headings

# Heading 1
## Heading 2
### Heading 3
#### Heading 4

## Paragraphs and emphasis

Documentation should be easy to read. Use **bold** for emphasis, *italic* for terms, and `inline code` for identifiers. Combine them when needed: ***bold italic***.

## Lists

Unordered:

- First item
- Second item
  - Nested item
  - Another nested item
- Third item

Ordered:

1. Step one
2. Step two
3. Step three
